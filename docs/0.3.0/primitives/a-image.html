<a-image>

Note: This documentation is for the old 0.3.0 version of A-Frame. Check out the documentation for the current 0.8.0 version

The image primitive displays an image on a flat plane. It is an entity that prescribes the geometry with its geometric primitive set to plane.

Fine-Tuning

Ensuring that the image is not distorted by stretching requires us to appropriately set the width and height.
