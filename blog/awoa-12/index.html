A Week of A-Frame 12

May 6, 2016

What's up with A-Frame for the week of Apr 29, 2016 to May 6, 2016.

A-Frame is a web framework for building VR experiences.

Projects

  • anj built Moon Bowling —

    Bowling shoes and gravity provided

  • EricaLayton built Moss March —

    A foggy swamp

  • EricaLayton built Space Blob Towers —

    A pointy alien world

  • EricaLayton built Carnival Globe Trees —

    Watch the globe-shaped trees rotate to the relaxing music

  • anj built A-Brush —

    Draw in 3D space with WASD + Cardboard cursor

  • georational built HelloWorldVR —

    Stonehenge after dark

  • rudy built May the 4th Be With You —

    May the Fourth Be With You. A-lways.

  • andreasplesch built Faceset Component —

    Prototype of a faceset component

  • gasolin built href Component —

    A component for linking to other pages when an entity is clicked. Note that VR mode will not be retained until WebVR 1.0 API is rolled out

  • anj built World —

    Explore 360° photos using globe navigation

  • wizgrav built Post-Processing Prototype —

    An example of post-processing scenes in A-Frame.
