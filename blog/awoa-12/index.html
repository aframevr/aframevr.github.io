A Week of A-Frame 12

May 6, 2016

Notable Contributions

  • JohnRodney fixed animation 'from' attribute when using dot property syntax
  • jsando fixed building bundles when using npm 3

Media

  • @joshcarpenter talked on the "Voices of VR" podcast about VR on the Open Web with A-Frame and WebVR
  • @vincent_munoz wrote about WebVR mockups using A-Frame
  • @bnolan wrote about lightmapping on the server
  • @bnolan wrote about exporting A-Frame-based scenes in SceneVR
  • @bnolan wrote about better selection tools in the SceneVR editor
  • @bnolan wrote about using Thingiverse files in A-Frame

Projects

  • anj built Moon Bowling — Bowling shoes and gravity provided
  • @EricaLayton built Moss March — A foggy swamp
  • @EricaLayton built Space Blob Towers — A pointy alien world
  • @EricaLayton built Carnival Globe Trees — Watch the globe-shaped trees rotate to the relaxing music
  • anj built A-Brush — Draw in 3D space with WASD + Cardboard cursor
  • georational built HelloWorldVR — Stonehenge after dark
  • @rudy built May the 4th Be With You — May the Fourth Be With You. A-lways.
  • andreasplesch built Faceset Component — Prototype of a faceset component
  • gasolin built href Component — A component for linking to other pages when an entity is clicked. Note that VR mode will not be retained until WebVR 1.0 API is rolled out
  • anj built World — Explore 360° photos using globe navigation
  • wizgrav built Post-Processing Prototype — An example of post-processing scenes in A-Frame.
