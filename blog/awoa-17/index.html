A Week of A-Frame 17

Jun 10, 2016

What's up with A-Frame for the week of Jun 3, 2016 to Jun 10, 2016.

A-Frame is a web framework for creating virtual reality experiences with HTML.

The entire Mozilla VR team, Don McCurdy, and DrawVR will be hacking on WebVR demos for a straight week in London. Forecast calls for a storm next week.

Projects

  • ideaspacevr built Particle System Component —

    A particle system component to create large numbers of very small objects. Used to create things such as rain, dust, snow, fire, water.

  • iandeboisblanc built VR Wiki Museum —

    View any Wikipedia page in VR mode!

  • richardanaya built WebVR Controller Component —

    Component to integrate HTC Vive controllers!

  • uniformtweets built Sketch VR —

    A Tilt Brush alternative for Cardboard Doodle on paper, take a picture, and view it in VR!

  • SebScholl built Virtual Reality Music Player —

    WebVR music player in space using the Spotify API. (code)

  • SebScholl built Weather Room —

    What’s the forecast for WebVR? Sunny with no chance of rain. (code)

  • fernandojsg built Bullets (Teaser) —

    Ever wanted to feel like Lara Croft?

  • fernandojsg built A-Frame Painter (Update) —

    Using geometry instead of screenspace lines.

  • Decipher built A-Frame + Drupal —

    Drupal module for integration with A-Frame.

  • LeBinoclard built CinemVR Design Prototype —

    Design prototype of a VR cinema.
